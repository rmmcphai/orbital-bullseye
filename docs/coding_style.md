# Coding Style (v1)

## Indentation
- **4 spaces per indent level**
- **No tabs** (tabs are treated as 4 spaces if encountered)

Enforced by:
- `.editorconfig`
- `.clang-format`

## Formatting
- C/C++ formatting uses `clang-format` with the repo `.clang-format`.
- Do not commit large reformat-only diffs unless explicitly part of a correction.

## Line endings
- LF
